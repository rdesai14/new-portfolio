# Rishi Desai — Portfolio

A cinematic, single-page developer portfolio built with React, Tailwind CSS, and Lucide React. Georgia Tech themed with navy and gold aesthetics.

## Features

- Cinematic boot sequence with shattering text entry animation
- Hero section with self-typing specialization matrix
- Interactive Bento Grid with cursor-reactive glow cards
- Fluid project showcase with category filtering
- Expandable experience timeline with tech-spec dropdowns
- Functional mini IDE/Terminal to explore resume via commands

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Live Site

Deployed via GitHub Pages: [https://rdesai14.github.io/new-portfolio/](https://rdesai14.github.io/new-portfolio/)

Pushes to `main` automatically rebuild and deploy through GitHub Actions.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 6
- Tailwind CSS 3
- Lucide React

## Terminal Commands

Once loaded, scroll to the terminal section or click **Launch Terminal** and try:

- `help` — list all commands
- `about` — personal introduction
- `skills` — technical skill matrix
- `experience` — work history
- `projects` — featured projects
- `contact` — contact info
- `neofetch` — system info banner
